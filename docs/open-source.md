# Open Source & Trademarks

## What Is Open Source

Hyphi is committed to open hardware and open software where it matters most — the protocol and the app that runs it.

### OWLS Protocol

The **Open Wireless Lighting Standard** is fully open. The specification — services, UUIDs, wire formats, mesh topology — is published here and free to implement on any hardware.

- Anyone can build a device that speaks OWLS
- Anyone can build an app that controls OWLS devices
- No license required, no royalties

### Hyphi Hub

**Hyphi Hub** (the PWA companion app) is open source.

- Source: [github.com/skyfly200/hyphi-hub](https://github.com/skyfly200/hyphi-hub)
- Fork it, extend it, build your own controller
