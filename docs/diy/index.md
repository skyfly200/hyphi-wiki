# DIY Build Guides

<StatusBadge type="wip" />

Step-by-step guides for assembling Hyphi hardware. Hyphi boards come pre-assembled — these guides cover wiring your LEDs, connecting power, and getting everything running.

## Before You Start

These guides assume basic electronics comfort — if you've soldered wires or connectors before, you're set. If not, read [Soldering Tips](./soldering) first.

::: tip Pre-assembled boards
Hyphi PCBs ship with all surface-mount components already soldered. Your job is wiring: LEDs, battery, connectors, and enclosure.
:::

## What You'll Need

- Soldering iron with fine tip (temperature-controlled)
- Solder — 63/37 or 60/40, 0.5mm diameter
- Flux pen
- Tweezers (ESD-safe)
- Multimeter
- USB-C cable (data-capable)
