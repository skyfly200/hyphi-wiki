# Tools & Materials

<StatusBadge type="stable" />

Recommended tools for building Hyphi hardware. Links are suggestions — source from wherever makes sense for you.

## Essential Tools

| Tool | Recommendation | Notes |
|------|---------------|-------|
| Soldering iron | Hakko FX-888D, TS80P, or Pinecil | Temperature-controlled is mandatory |
| Solder | Kester 63/37 0.5mm | Lead-free is harder — use 63/37 for learning |
| Flux | MG Chemicals 835 flux pen | Apply before soldering pads |
| Tweezers | Hakko CHP 5-SA ESD | Fine-tip, ESD-safe |
| Multimeter | Any with continuity beeper | Essential for debugging shorts |
| Helping hands / PCB holder | PanaVise or generic | Keeps boards stable while soldering |
| USB-C cable | Data-capable cable | Many cables are charge-only — test yours |

## For SMT / 0402 Work

| Tool | Notes |
|------|-------|
| Hot air station | Quick 861DW or similar — essential for 0402 and ICs |
| Solder paste | Sn63Pb37 T4 paste — keep refrigerated |
| PCB stencil | Order with your JLCPCB PCB for consistent paste application |
| Microscope or loupe | 10–40× — makes 0402 work much less frustrating |
| ESD mat | Protects ESP32 and sensitive ICs |

## Consumables

- Solder wick (desoldering braid)
- Isopropyl alcohol 90%+ (flux cleanup)
- Cotton swabs / lint-free wipes
- Kapton tape (masking during hot air work)
- Heat shrink tubing (various sizes for wire work)

## Test Equipment (Optional but Helpful)

| Tool | Use |
|------|-----|
| Oscilloscope | Debug WS2812B data signal integrity |
| USB power meter | Measure real current draw |
| Logic analyzer | Decode BLE / UART debug output |

## Sourcing

- **Components:** JLCPCB Parts Library (ships with PCB order), AliExpress, Mouser, Digi-Key
- **PCBs:** JLCPCB (primary), PCBWay
- **Tools:** Amazon, Adafruit, SparkFun, AliExpress
